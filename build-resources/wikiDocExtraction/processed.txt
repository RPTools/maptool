strformat	Returns a string formatted based on the escape sequences in the passed string and optional arguments. <p>The format string can contain special instructions that are introduced with the <code>%</code> symbol. There are two different approaches to using <code>strformat</code>. <p>(This function is directly implemented by Java. You can find more details on the syntax of the format <a class="external text" href="https://docs.oracle.com/javase/6/docs/api/java/util/Formatter.html#syntax" rel="nofollow">here</a>. In particular, that page documents the <b>argument_index</b> modifier which isn't specified here, and MapTool numbers are converted to <code>BigInteger</code>'s when formatting integers and <code>BigDecimal</code>'s when formatting floating point values.) <h4><span class="mw-headline" id="Simple_words">Simple words</span></h4> <p>Simply said, this means that it facilitates creating a string (sentences) which consists of variables or string operations. It's especially useful if you want to store the string into a variable first. The most common method is concatenation: <div class="mw-highlight mw-content-ltr" dir="ltr"><pre> [h:target = "Orc"] [h:tok = token.name] [h:hit = 2d6] [h:output = tok + "hits the " + target + " for: " + hit + "wounds."] [r:output]</pre></div> <p>Using <code>strformat</code>, this becomes (just replacing line 4): <div class="mw-highlight mw-content-ltr" dir="ltr"><pre> [h:output = strformat("%{tok} hits the&#xA0;%{target} for:&#xA0;%{hit} wounds.")]</pre></div> <p>Or the other method, if you do not wish to create variables first: <div class="mw-highlight mw-content-ltr" dir="ltr"><pre> [h:target = "Orc"] [h:output = strformat("%s hits the&#xA0;%{target} for: %d wounds.", token.name, 2d6)]</pre></div> <p>So why would you use this over the concatenation method? Four reasons: <ol><li>it's cleaner (easier to read),<li>it's faster (though barely noticeable),<li>you can embed the whole operation inside a function call, and<li>it has lots of formatting options.</ol> <h4><span class="mw-headline" id="Formatting_by_Data_Type">Formatting by Data Type</span></h4> <p>The most general use of <code>strformat</code> is to pass a format string first, followed by a list of parameters. Each parameter is matched against the corresponding marker in the format string. Any characters in the format string that are not markers are treated as literal text to be output. <pre>&#xA0;%(+0x </pre> <p>The first character is the percent sign (<code>%</code>).
