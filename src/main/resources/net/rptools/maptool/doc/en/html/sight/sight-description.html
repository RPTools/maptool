Sight Properties

The sight properties define how a token can see

Name

This is the name of the light source

Shape

The shape of the vision, this can be one of

  • Circle
  • Cone
  • Square
  • Grid

Distance

The distance that the vision extends to, if this is 0 then it uses the vision distance from the map.

From Edge of Token

If this is checked then the distance is measured from the edge of the token, otherwise it is measured from the center of the token.

Arc

Specifies how large the field of vision should be for conical vision (in Degrees).

Offset

Specifies how many degrees offset conical vision should be from the tokens facing.

Light Source Multiplier

The multiplier value to add to light sources when using this vision type. For example if a low light vision doubled the distance of light source then this value would be 2.

Personal Lights

As well as the main sight properties you can define personal light sources that tokens with this vision have. A personal light is only "seen" by the token in question. Personal light sources can be used for things like

  • Darkvision where the token does not need any "normal" light source to see
  • Light sources that only the token can see
  • To handle any rules such as a player can always determine what is 5' away via sight/touch

A token can never see the personal light source of another token even if they are using the same vision type.

Personal Light Distance

The distance that the personal light source for the token extends.

Personal Light Color

The color of the personal light source for the token.

Personal Light Lumens

The lumens value for the personal light source. The lumens determine the strength of the light source and are compared against lights with a negative value (darkness) to determine if they light up the darkness or not. Seet the light tab for more information about lumens
